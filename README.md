# Auto Spam Appeal (ASA) - 自动误封申诉 Mod

这是一个专为 Minecraft 1.21.4 设计的 Fabric Mod，旨在自动化特定服务器中的“误封申诉”流程。它可以自动完成从掉线重连、寻找申诉点、模拟交互到发送申诉理由的全过程。

## 🚀 主要功能

1.  **自动重连 (Step 1)**：检测到被封禁（Banned/Spam）提示时，5 秒后自动尝试重连。
2.  **智能环境识别 (Step 2)**：自动检测玩家是否处于申诉区域。
    *   **坐标判定**：X: 12.5 (+/-5), Z: 10.5 (+/-5), Y: -59。
    *   **方块判定**：玩家脚下 3x3 范围内必须有 **5 个以上** 目标方块（各种木板、闪长岩、安山岩等）。
3.  **自动位移与攻击 (Step 3)**：识别环境后，自动 TP 到 `14, -59, 14` 并左键攻击最近的假人（Bot）以触发系统。
4.  **GUI 自动点击 (Step 4 & 5)**：
    *   自动在“反馈系统”菜单中点击 **红石块**。
    *   自动在“选择子类型”菜单中寻找并点击名为“**自述申诉**”的物品。
5.  **Title 识别发包 (Step 6)**：监测屏幕上的“醒醒！”提示，并在 1 tick 内自动发送预设的申诉理由。
6.  **安全退出 (Step 7)**：检测到服务器反馈“反馈成功”或“反馈失败”后，等待 2 秒自动断开连接。
7.  **输入保护**：自动化执行期间，键盘（除 ESC 外）和鼠标将被锁定，防止手动干扰。

## ⌨️ 按键绑定

*   **`O` 键**：打开 ASA 设置界面。
*   **`P` 键**：强制开始申诉流程（跳过环境检测，直接进入 TP 阶段）。
*   **`ESC` 键**：在自动化运行期间，按下 ESC 可立即停止所有动作，并进入 5 秒冷却期。

## ⚙️ 配置说明

按 **`O`** 键可进入设置 GUI：
*   **ASA 状态**：全局开关。
*   **调试信息**：是否在 Action Bar（屏幕中心下方）显示当前执行的步骤状态。
*   **申诉理由**：设置你要自动发送的文字内容（保存后即时生效）。

> **注意**：所有配置会自动保存在 `.minecraft/config/autospamappeal.json` 中，下次启动游戏无需重新设置。

## 🛠️ 安装要求

1.  **Minecraft 版本**：1.21.4
2.  **加载器**：Fabric Loader (建议 0.16.9+)
3.  **必要依赖**：Fabric API
